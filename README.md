# Logistic Regression - Binary Classification

## 📌 Objective
To build a binary classification model using Logistic Regression and evaluate its performance using various metrics.

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Dataset
Breast Cancer Wisconsin Dataset (built-in from Scikit-learn)

## 🚀 Steps Performed
1. Imported required libraries
2. Loaded dataset using sklearn
3. Converted dataset into DataFrame
4. Split data into training and testing sets
5. Standardized features using StandardScaler
6. Trained Logistic Regression model
7. Made predictions on test data
8. Evaluated model using:
   - Confusion Matrix
   - Precision & Recall
   - Classification Report
   - ROC-AUC Score
9. Plotted ROC Curve

## 📈 Results
- The model achieved high accuracy
- Precision and Recall values are strong
- ROC-AUC score is close to 1, indicating excellent model performance

## 🔍 Observations
- Logistic Regression works well for binary classification problems
- Proper feature scaling improves model performance
- Model effectively distinguishes between classes

## ✅ Conclusion
The Logistic Regression model successfully classified the data with high performance. It is a simple yet powerful algorithm for binary classification tasks.
